Sobre o projeto

Este é um projeto voluntário e tem como objetivo espelhar os dados de tramitação parlamentar no Brasil, inicialmente da câmara e senado, extensível a outras casas legislativas estaduais e municipais.

Por que?

Os dados públicos que permitem a sociedade acompanhar e cobrar os governantes é disponível por meio de algumas ferramentas e serviços. No entanto, não há meio de se obter acesso a estas informações de forma automatizada e indolor para desenvolvimento de aplicações para cruzamento e indexação dos dados

Com quem eu falo, por favor?

Você pode começar a se envolver acessando as discussões do projeto parlamentoaberto.org.

Quem apoia?

Por enquanto, o projeto tem apoio do estudio cromática.

Todo o código fonte deste site incluindo os mecanismos de raspagem (scraping) que estão sendo utilizados estão disponíveis no git hub sob licença GPLv3
